== Description ==
Adds a drop-down to the admin bar with links to the current page in your other site/s.

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/twin-sites` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Appearance->Menus screen to configure the plugin
